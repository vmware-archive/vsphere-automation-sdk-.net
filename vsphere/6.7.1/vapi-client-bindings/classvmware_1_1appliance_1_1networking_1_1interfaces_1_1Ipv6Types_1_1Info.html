Detailed Description

The Ipv6.Info class provides the structure that defines an existing IPv6 configuration on a particular interface.

This structure is read only. This class was added in vSphere API 6.7.

string vmware::appliance::networking::interfaces::Ipv6Types::Info::GetDefaultGateway ( ) [inline]

The default gateway for static IP address assignment.

This configures the global IPv6 default gateway on the appliance with the specified gateway address and interface. This gateway replaces the existing default gateway configured on the appliance. However, if the gateway address is link-local, then it is added for that interface. This does not support configuration of multiple global default gateways through different interfaces. This attribute was added in vSphere API 6.7.
