string vmware::appliance::networking::interfaces::Ipv4Types::Config::GetDefaultGateway ( ) [inline]

The IPv4 address of the default gateway.

This configures the global default gateway on the appliance with the specified gateway address and interface. This gateway replaces the existing default gateway configured on the appliance. However, if the gateway address is link-local, then it is added for that interface. This does not support configuration of multiple global default gateways through different interfaces. This attribute was added in vSphere API 6.7.
