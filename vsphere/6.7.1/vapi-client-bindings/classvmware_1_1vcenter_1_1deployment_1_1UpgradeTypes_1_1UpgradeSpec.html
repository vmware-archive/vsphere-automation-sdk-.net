vmware.vcenter.deployment.HistoryMigrationSpec vmware::vcenter::deployment::UpgradeTypes::UpgradeSpec::GetHistory ( ) [inline]

Determines how vCenter history will be migrated during the upgrade process.

vCenter history consists of:

  • Statistics
  • Events
  • Tasks

By default only core data will be migrated. Use this spec to define which part of vCenter history data will be migrated and when. This attribute was added in vSphere API 6.7.
